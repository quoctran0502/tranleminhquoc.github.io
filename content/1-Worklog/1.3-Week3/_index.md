* Learn and practice identity and access management with AWS IAM, virtual server deployment with EC2, and object storage with Amazon S3.

### Week 3 Achievements:

* Understood authorization mechanisms and key concepts of Users, Groups, Roles, and Policies in AWS IAM.

* Successfully created and configured access permissions for IAM user accounts using managed IAM Policies.

* Practiced launching and configuring virtual EC2 instances using appropriate AMIs, Key Pairs, and Security Groups.

* Learned how to connect to instances via SSH/EC2 Instance Connect, monitor status, stop, and restart EC2 instances.

* Got familiar with Amazon S3: successfully created S3 Buckets, uploaded/downloaded and managed objects within the buckets.
