### Week 4 Achievements:

* Understood Block Storage concepts and the operational mechanics of Amazon EBS.

* Practiced creating, attaching, and verifying the storage status of EBS Volumes on EC2 instances.

* Mastered Snapshots and Backups: successfully created EBS Snapshots, restored Volumes from Snapshots, and compared AMIs with Snapshots.

* Researched Amazon AMIs: created an AMI from a running EC2 instance and successfully launched a new instance from that AMI.

* Understood Elastic IPs (static Public IPs), associated an Elastic IP with an EC2 instance, and verified connection.

* Studied the EC2 User Data mechanism and automated Apache web server installation upon instance launch.
