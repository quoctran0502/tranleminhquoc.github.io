This section introduces the technical blogs I have written after studying AWS documentation and official AWS Blog articles. These blogs summarize key concepts, AWS services, reference architectures, and practical applications related to cloud computing and security on AWS.

### [Blog 1 - Rethinking Binary Asset Storage on AWS with Lore](3.1-Blog1/)

This blog explains how **Lore** rethinks binary asset storage by adopting a content-addressed storage architecture on AWS. It discusses the limitations of traditional binary asset management and demonstrates how services such as Amazon S3, Amazon EC2, Amazon ECS, AWS Cloud Map, and Amazon DynamoDB work together to reduce duplicate data, optimize storage costs, and improve scalability for large-scale game development and digital media projects.

### [Blog 2 - Preventing Data Exfiltration on AWS with Egress Controls for Cloud Workloads](3.2-Blog2/)

This blog introduces the concept of **Data Exfiltration** and explains why controlling outbound network traffic (Egress Traffic) is essential for cloud security. It also describes how AWS services including AWS Network Firewall, Amazon Route 53 Resolver DNS Firewall, Amazon GuardDuty, AWS Security Hub, and Data Perimeter controls can be combined to build a multi-layered security architecture that helps detect and prevent unauthorized data leakage.

### [Blog 3 - Automating Identity Lifecycle and Enhancing Security with AWS Directory Service APIs](3.3-Blog3/)

This blog explores the newly introduced **AWS Directory Service Data APIs** for AWS Managed Microsoft Active Directory. It explains how organizations can automate identity lifecycle management and security response workflows by integrating services such as Amazon GuardDuty, Amazon EventBridge, AWS Step Functions, AWS Systems Manager, and Amazon SNS. The blog also highlights the benefits of security automation in improving operational efficiency and strengthening enterprise identity management.
